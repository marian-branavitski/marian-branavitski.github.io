All my projects

This page contains the screenshots of some of my projects. You can find the source code for all my projects on my git hub.

    • AI Assistant

    This is project in work and is quite a way from being considered fully finished, however the foundation is already complete. Written in python 3 and using sqlite for storing previously asked questions this projects aims to recreate a personal assistant like Google or Siri. It uses many other AI agents such as conqui ai to generate the voice of the assistant. Vosk for voice recognition and Ollama API to allow using local LLMs which keeps this assistant completely secure and private as all is stored localy.

    • Kinowhere

    Kinowhere is a film based web site similar to netflix. It has CRUD functionality for the films but only for super users. Regular users can only view films data and save them to their database. Uses PHP framework Codeigniter for backend and MySQL as it's database.

    • Song Player

    Project written in Java uses Swing for GUI. This is my A-level project it uses text documents to store user created playlists as well as their credentials for authentication. All sensetive data is encrypted using custom written encryption algorithm, passwords are hashed using custom written hash encryption algorithm.

    • Snake Game

    This is a simple 2D snake game written on python. It relies on the library pygame to render the graphics and take constant input from the user to allow controling the snake.

    • Toys By Svetlana

    Toys By Svetlana is a blog type website which allows the owner to create, delete and edit toys and news articles. The website allows to send emails through the contact page. This website uses PHP framework Codeigniter and MySQL for working with database. Like kinowhere this website has a user data base.
